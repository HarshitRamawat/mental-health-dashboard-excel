# Mental Health Survey Excel Dashboard

This project is an interactive Excel dashboard created using a mental health survey dataset.

## Dashboard Features

- KPI cards for quick insights
- Interactive slicer-style filters
- Clean teal and dark theme
- Pivot-based charts
- Department, region, gender, and work model analysis
- Easy-to-understand visual layout

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Dashboard Design

## File Included

- `mental_health_dashboard_CLEAN_FIXED.xlsx`

## Purpose

The purpose of this project is to analyze mental health survey data and present the insights in a clean, attractive, and interactive Excel dashboard.
